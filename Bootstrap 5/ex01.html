Configurar Projeto no eclipse

  1. Selecionar Java EE
  2. Selecionar Dynamic Web Project
  3. Selecionar servidor (tomcat 9)
  4. Indicar caminho do apache
    5. Deve ser extraido e colado na raíz do eclipse

Criar páginas no projeto

  1. Páginas ficam na pasta /webapp
  2. HTML é escrito em arquivos do tipo .jsp
  3. Criar arquivo do tipo .jsp na pasta /webapp
  4. Para executar, selecionar o navegador em window > web browser

Executar projeto

O código em java é escrito dentro do html por meio de tags transparentes: <% %>. Essas tags não são renderizadas no navegador.

Dentro das tags transparentes, a função out.print() imprime valores no navegador.
